> 「在平面上，直線與圓的相遇，藏著解析幾何的所有秘密。」

## 基本觀念

平面上的圓可以用標準方程式 \((x-a)^2 + (y-b)^2 = r^2\) 表示，其中 \((a,b)\) 為圓心，\(r\) 為半徑。直線常見的表示方式為斜截式 \(y = mx + c\) 或一般式 \(Ax + By + C = 0\)。

要判斷直線與圓的相對位置，可將直線方程代入圓的方程，化為一元二次方程：

\[
  (1 + m^2)x^2 + 2(m(c-b) - a)x + (a^2 + (c-b)^2 - r^2) = 0.
\]

- 判別式 \(\Delta > 0\)：兩個交點（相交）。
- 判別式 \(\Delta = 0\)：一個交點（相切）。
- 判別式 \(\Delta < 0\)：沒有交點（相離）。

若使用一般式直線，只需將直線代入並檢查二次式的判別式即可。

## 與半徑的關係

若直線以一般式表示，直線到圓心的距離為

\[
  d = \frac{|Aa + Bb + C|}{\sqrt{A^2 + B^2}}.
\]

比較 \(d\) 與半徑 \(r\)：

- \(d < r\) 時相交。
- \(d = r\) 時相切。
- \(d > r\) 時相離。

## 範例：求交點座標

考慮圓 \((x-1)^2 + (y+2)^2 = 25\) 與直線 \(3x - 4y - 20 = 0\)。

1. 先算距離：\(d = |3 \cdot 1 - 4 \cdot (-2) - 20| / 5 = 3\)，因為 \(r = 5\)，故相交。
2. 將直線改寫為 \(y = \frac{3x-20}{4}\) 代入圓：

\[
  (x-1)^2 + \left(\frac{3x-20}{4} + 2\right)^2 = 25.
\]

3. 化簡得 \(25x^2 - 170x + 289 = 0\)，解得 \(x = 4\) 或 \(\frac{289}{25}\)。
4. 代回直線求 \(y\)，可得交點為 \((4, -2)\) 與 \((\frac{289}{25}, -\frac{47}{25})\)。

## 應用練習

- 將一條直線 \(y = x + 1\) 與圓 \(x^2 + y^2 = 10\) 聯立，判斷相對位置並求交點。
- 設計一條通過點 \((2,3)\) 且與圓 \((x-3)^2 + (y-1)^2 = 13\) 相切的直線。
- 探討直線 \(x = k\) 與單位圓 \(x^2 + y^2 = 1\) 的交點數量，並討論參數 \(k\) 的條件。
